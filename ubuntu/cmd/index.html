    Ubutnuコマンド

    2019/04/27

    頻出コマンド

    最低限覚えきたいものたち。

    ls ディレクトリ・ファイルの情報の確認

    1
    ls
    「list segments」の略らしい。知らなかった。
    オプションをつけると追加で情報が得られる。 ls -la は割とよく使っている。

    • 隠しファイルを確認する 
      1
      ls -a
    • 権限と最終更新日の確認 
      1
      ls -l

    cd カレントディレクトリの移動

    カレントディレクトリは現在作業しているディレクトリのこと。

    1
    cd 相対パスor絶対パス

    • ホームディレクトリに戻る 

      1
      cd

    • 親ディレクトリに移動 

      1
      cd ..

    pwd 現在のディレクトリの表示

    1
    pwd

    apt, apt-get

    • パッケージのinstall 
      1
      sudo apt install パッケージ名
    • パッケージの削除 
      1
      sudo apt remove パッケージ名
    • 利用可能なパッケージ一覧の更新 
      1
      sudo apt update
      追加削除を行うときはにしておくとよい。
    • パッケージの更新 
      1
      sudo apt upgrade
    • ヘルプ 
      1
      apt -h
      他にもできることはいっぱいあるので困ったらヘルプを見るのが無難。
    apt -h の最終行

    この APT は Super Cow Powers 化されています。
    スーパー牛さんパワーとは…

..."Have you mooed today?"...

    aptitude -h の最終行

    この aptitude にはスーパー牛さんパワーなどはありません。

    興味があれば 

    1
2
3
4
5
6
7
    aptitude moo
aptitude -v moo
aptitude -vv moo
aptitude -vvv moo
aptitude -vvvv moo
aptitude -vvvvv moo
aptitude -vvvvvv moo
    を上から順番にどうぞ。
    aptitude がない場合はsudo apt install aptitude でできるようになります。

    apt-get か aptか

    多くのサイトでapt-get コマンドが使われているのを見るが、apt と置き換えて良さそう。

    「apt-get」はもう古い？新しい「apt」コマンドを使ったUbuntuのパッケージ管理

    aptコマンドでやれることの説明も書かれているのでaptで困ったら見てみると良いかも。

    リモートマシンにsshでログインする。 自分がやったときは何故か一度はじかれた。原因不明。もう一回やったらできたので謎。

      1

    パスワードの変更。 現在のパスワードと，新しいパスワードx2回要求される

    scp 転送するファイル(ユーザ名@aaa.bb.ccc.XX:~/(転送するファイル))  転送先
