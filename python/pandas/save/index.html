    書き込み

    Pandasを利用するときには以下を記述しておく。 

    1
    import pandas as pd

    書き込み(保存)

    データフレームをcsvなどに保存できる。 基本。 

    1
    df.to_csv("hoge.csv")
    保存をする際にエンコーディングなどの指定が可能。 
    1
    f.to_csv("hoge.tsv", encoding="shift_jis",sep='\t',index='False')
    その他オプション sep : 区切り文字の指定。デフォルトは , index : 行名を入れるかどうか。デフォルトは True

    Windows環境でExcelを開く予定がある場合、デフォルトだと文字化けを起こす。 タブ区切りUTF-16を指定するとうまくいった気がする。 

    1
    df.to_csv("hoge.csv",encoding="UTF-16",sep='\t')

    Excelファイルとして保存

    excelへの書き込みにも対応している。 csvでエンコードをどうにかしてExcelで見るよりもこっちのほうが圧倒的に楽。

    1
    df.to_excel('document.xlsx', sheet_name='Sheet1')

    ただし、openpyxl が必要。

    1
    pip install openpyxl

import pandas as pd
data = {"id" : id_list,
        "title" : title_list}
w_df = pd.DataFrame(data)
w_df.to_excel('document.xlsx', sheet_name='Sheet1')
print("excelへの書き込みを完了しました")
w_df.to_csv('document.csv')
print("csvへの書き込みを完了しました")
