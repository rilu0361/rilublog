コードブロック 
==============

書き方1 ::

    書き方1 ::

        for a in range(10):
            print(a)

        print(a)

    コードは前後一行を空行にしてインデントする

| 「::」の前に空白を入れると「:」は非表示になる。
| それ以外では「:」が表示される。

書き方２

.. code-block:: none
    :linenos:
    :caption: コードブロックの書き方
    :emphasize-lines: 2,3,4

    .. code-block :: none
        :linenos:
        :caption: コードブロックの書き方
        :emphasize-lines: 1,3

        for a in range(10):
                print(a)

            print(a)

オプション

* ``code-block:: python`` : pythonを書いている宣言
* ``linenos`` : 行番号をつける
* ``emphasize-lines`` : 特定の行のハイライト
* ``caption`` : キャプション
* ``name`` : 参照の際の名前

画像の挿入
==========
```rest
.. image:: ./image/title.png
    :align: center
```

* ``align`` : どこ揃えにするか.
    ``top`` , ``middle`` , ``bottom`` , ``left`` , ``center``, ``right``
* ``height`` : 高さ
* ``width`` : 幅
* ``scale`` : 100%で
* ``alt`` : 代替えテキスト
