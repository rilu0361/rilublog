## 英語などのパース

基本的にスペースでパースすれば良いはず。  
最初の大文字などが別語彙になってしまうため、全て小文字に統一する。  
また、ピリオドやコンマが単語ごとにくっついていると問題なので離す処理をする。  

```py
text = "This is a pen."
text = text.lower() #大文字を小文字に変換する
text = text.replace("."," .") # ピリオドの前に空白を挿入
words = text.split() # 空白(スペース)がある度に分割
print(words)
```

その他、単語を基本形(原形)に統一するなどしてみるといいのかも。
