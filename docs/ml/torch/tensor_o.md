# テンソル参照

## 参照
numpyと同様に参照可能。

```py
import torch
x = torch.randn(2,3)
tensor([[ 0.5497,  0.2352,  0.4812],
        [ 0.8624, -1.2387, -0.3832]])
```

(1行目以降)
```py
x[1:]
```

- 結果
```sh
tensor([[ 0.8624, -1.2387, -0.3832]])
```

(1列目)
```py
x[:,1]
```

- 結果
```sh
tensor([ 0.2352, -1.2387])
```

## 値の取得

1要素のテンソルをPythonで扱える数値として取得可能.

```py
>>> x[0,0]
tensor(-0.2520)

>>> x[0,0].item()
-0.2519831955432892
```
